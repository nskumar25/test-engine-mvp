# Assessment Test Engine MVP

Student-facing web assessment MVP with:

- JSON-based assessment input
- MCQ questions
- Embedded question images
- Question navigation grid
- Calculator toggle by assessment configuration
- Whiteboard-style scratch pad
- Silent copy/selection friction
- Submission evaluation
- Browser IndexedDB result storage by student

## Run Locally

```powershell
node server.js
```

Open:

```text
http://127.0.0.1:5173/
```

## Current Input

The app currently loads:

```text
input/pre-test-for-demo.json
```

Images are stored in:

```text
input/assets/pre-test-for-demo/
```
